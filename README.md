# Portfolio Website
This is my portfolio website built from scratch to showcase my projects and technical skills. 

Through this project, I independently learned React and modern JavaScript while exploring frontend development. The website features a responsive design for both desktop and mobile, a dedicated projects page, and dynamic scroll-based interactions.

## Link to portfolio
https://hayleyfar-portfolio.vercel.app/

## Installation & Setup

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/hayley-far/hayleyfar-portfolio.git

# Navigate into the project folder
cd portfolio-react-app

# Install dependencies
npm install

# Start the development server
npm run dev
